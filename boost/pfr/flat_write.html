Function template flat_write

boost::pfr::flat_write — Writes flattened POD value to out

Synopsis

// In header: <boost/pfr/flat/io.hpp>


template<typename Char, typename Traits, typename T> 
  void flat_write(std::basic_ostream< Char, Traits > & out, const T & value);

Description

Note: If reflecting structures with enums may reinterpret_cast enum to a reference to underlying type.

Example: 

struct my_struct { int i, short s; };
my_struct s{12, 13};
flat_write(std::cout, s); // outputs '{12, 13}'
